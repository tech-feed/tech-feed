Gerando autonomia para comunidades

#### Projeto desenvolvido para o desafio Fome Zero do Call For Code 2021
_____________________________________________________

🎯 Objetivo: Dar autonomia para comunidades criando agricultores sustentáveis.

📢 Diferenciais:

- Simplicidade de implementação;
- Escalável;
- Redução de custos;
- Aumento da produtividade;
- Otimização de recursos.

💡 Como funciona:

<img alt="fluxo" src="https://media.discordapp.net/attachments/867195120340041738/868712192730607646/fluxo.png"  width=100%%/>

🚀 Vamos começar!

 - Clonar o repositório:
 > git clone https://github.com/tech-feed/tech-feed.git

 - Instalar o node:
 > sudo apt install nodejs

 - Instalar os pacotes:
 > sudo npm install

 - Iniciar o servidor:
 > sudo npm start

 - Acesse o preview do <a href="https://web-chat.global.assistant.watson.cloud.ibm.com/preview.html?region=eu-de&integrationID=ca82185b-7356-487e-a583-e2935c0d8dee&serviceInstanceID=4f00b77f-ce96-42ff-942f-9fcf9d6ac98d">feedBOT</a>
